# Easy BMI

A simple and straightforward way to calculate body-mass-index (BMI).

## Features
* Calculate BMI using imperial (lbs & in) or metric (kg & cm) units.
* Display calculated BMI on a chart
* Store the last 10 results locally

## Languages
- Dart

## Technologies Used
- Flutter
